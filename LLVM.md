# How to Install LLVM

1. Go to LLVM's [download page](https://releases.llvm.org/download.html) and download & untar the binary using `wget` and `tar -xvf`
2. `cd` into the folder
3. `$ sudo cp -R * /usr/local`
4. Verify using `llvm` command
